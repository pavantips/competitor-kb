# 🎯 Meazure Learning — Competitive Intelligence RAG Chatbot

A free RAG-powered chatbot for the Solutions Consulting / RFP team to instantly query competitive intelligence on 11 exam/assessment platforms.

## Platforms Covered
Caveon · Surpass · Cirrus · Excelsoft (Saras) · ITS · Elsevier · Questionmark · Risr · Prolydian · ROC-P · CMS/AUTHORWise

---

## 🖥️ Local Setup (Development)

### 1. Clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/meazure-competitive-intel.git
cd meazure-competitive-intel
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```
> First run downloads the `all-MiniLM-L6-v2` embedding model (~80MB). Subsequent runs use the cached version.

### 3. Add your Anthropic API key
```bash
cp .env.example .env
# Edit .env and add your key:
# ANTHROPIC_API_KEY=sk-ant-...
```

### 4. Ingest the competitive analysis into ChromaDB
```bash
python ingest_data.py
```
This creates a `chroma_db/` folder with your vector database. Only needs to run once (or when you update the markdown docs).

### 5. Launch the app
```bash
streamlit run app.py
```
Open http://localhost:8501 in your browser.

## ☁️ Streamlit Cloud Deployment (FREE)

### 1. Push to GitHub
```bash
git add .
git commit -m "Initial competitive intel RAG app"
git push origin main
```
> **Important:** Include the `chroma_db/` folder in your commit so Streamlit Cloud has the pre-built vector DB. Add it to git if needed: `git add -f chroma_db/`

### 2. Connect to Streamlit Cloud
1. Go to [share.streamlit.io](https://share.streamlit.io)
2. Click **New app**
3. Select your repo, branch (`main`), and main file (`app.py`)
4. Click **Deploy**

### 3. Add your API key as a secret
In Streamlit Cloud → your app → **Settings → Secrets**, add:
```toml
ANTHROPIC_API_KEY = "sk-ant-your-key-here"
```

### 4. Done! ✅
Your app is live at `https://your-app-name.streamlit.app`

## 🔄 Updating the Knowledge Base
1. Edit `competitive_analysis.md` (or add new `.md` files)
2. Re-run `python ingest_data.py` to rebuild the vector DB
3. Push the updated `chroma_db/` folder to GitHub
4. Streamlit Cloud will auto-redeploy

---

## 🛠️ Switching to a More Powerful Model
In `app.py`, change:
```python
MODEL = "claude-haiku-4-5"       # Fast, cheap (~$5-10/month)
# to:
MODEL = "claude-sonnet-4-6"      # More nuanced answers (~$30-50/month)
```
